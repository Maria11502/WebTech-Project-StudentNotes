# StudentNotes - Project


**StudentNotes** is a web application designed to help students efficiently take, organize, and manage notes during university courses, seminars, and individual study sessions.


##  Description
StudyNotes is a Single Page Application (SPA) that helps students manage notes by:
- subjects
- tags
- keywords
- date
- attachments  
- and more.


## Minimal Functionalities

- **Authentication:**  
  Login using institutional email (`@stud.ase.ro`).

- **Notes Management:**  
  View, add, edit, delete notes.

- **Attachments:**  
  Add images or documents.

- **Organization:**  
  Sort/filter by subject, date, tags, keywords.

- **Sharing:**  
  Share notes with colleagues or study groups.

- **External Content:**  
  Add YouTube videos, online conference links, Kindle highlights.



##  Technology Stack

### Frontend

### Backend

### Database
